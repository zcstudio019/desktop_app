# 贷款助手 - 对话式前端设计规范

> 类似豆包/ChatGPT 的对话式 AI 助手界面
> 技术栈：React + Tailwind CSS + shadcn/ui + DeepSeek API

---

**消息气泡组件**:
```typescript
interface Message {
  id: string;
  role: 'user' | 'assistant';
  content: string;
  timestamp: Date;
  attachments?: Attachment[];  // 用户上传的文件
  actions?: MessageAction[];   // AI 消息的操作按钮

interface Attachment {
  id: string;
  name: string;
  type: string;
  size: number;
  status: 'uploading' | 'processing' | 'done' | 'error';
}

interface MessageAction {
  label: string;           // "保存到飞书" | "下载" | "查看详情"
  icon: ReactNode;
  onClick: () => void;
}
```

**用户消息样式**:

## 五、本地存储设计

### 5.1 存储结构

```typescript
// localStorage keys
const STORAGE_KEYS = {
  CHAT_HISTORY: 'loan_assistant_chats',      // 对话历史
  APPLICATIONS: 'loan_assistant_applications', // 申请表记录

  SETTINGS: 'loan_assistant_settings',        // 用户设置
};

// 对话历史
interface ChatHistory {
  id: string;
  title: string;           // 自动生成或用户编辑
  messages: Message[];
  createdAt: Date;
  updatedAt: Date;
}

// 申请表记录
interface ApplicationRecord {
  id: string;
  customerName: string;
  loanType: 'enterprise' | 'personal';
  content: string;         // Markdown 内容
  createdAt: Date;
  chatId?: string;         // 关联的对话ID

// 方案记录
interface SchemeRecord {
  id: string;
  customerName: string;
  matchCount: number;
  content: string;         // Markdown 内容
  createdAt: Date;
  chatId?: string;         // 关联的对话ID
}
```

```
单个对话最大消息数: 100 条

申请表最大保留: 100 个
方案记录最大保留: 100 个

超出限制时: 自动删除最早的记录
```

// POST /api/chat
interface ChatRequest {
  messages: {
    role: 'user' | 'assistant';
    content: string;

    type: string;

interface ChatResponse {
  message: string;

## 七、DeepSeek 集成

### 7.1 系统提示词

```
你是一个专业的贷款助手，可以帮助用户：
1. 提取各类贷款资料信息（征信、流水、财务数据等）
2. 生成贷款申请表
3. 匹配合适的贷款方案

当用户上传文件时，你需要：
1. 识别文件类型（企业征信/个人征信/企业流水/个人流水/财务数据/抵押物信息/水母报告/个人纳税公积金）
2. 提取关键信息并以结构化格式返回
3. 询问用户下一步操作

当用户要求生成申请表时，你需要：
1. 确认客户名称和贷款类型
2. 根据已有资料生成完整的申请表

当用户要求匹配方案时，你需要：
1. 根据客户资料和产品库进行匹配
2. 返回推荐的贷款方案

请始终保持专业、友好的态度。
```

### 7.2 意图识别

```typescript
// AI 返回的意图类型
type Intent = 
  | 'extract'      // 提取资料
  | 'application'  // 生成申请表
  | 'matching'     // 匹配方案
  | 'save'         // 保存到飞书
  | 'download'     // 下载文件
  | 'chat';        // 普通对话

// 根据意图触发不同的处理流程
function handleIntent(intent: Intent, data: any) {
  switch (intent) {
    case 'extract':
      // 调用 OCR + AI 提取
      break;
    case 'application':
      // 调用申请表生成
      break;
    case 'matching':
      // 调用方案匹配
      break;
    case 'save':
      // 调用飞书保存
      break;
    case 'download':
      // 触发文件下载
      break;
    case 'chat':
      // 普通对话回复
      break;
  }
}
```
